# 📋 ПОСТАНОВКА ЗАДАЧИ

**Название:** Улучшение страницы профиля — подписка, тарифы, оплата

**Цель:** Сделать страницу профиля информативной и конверсионной: пользователь должен чётко понимать свой текущий тариф, разницу между бесплатной и платной версией, стоимость, доступные способы оплаты.

**Проблема:** Текущий профиль содержал минимальную секцию подписки — просто статус текстом и одну кнопку «Оформить Premium» без деталей. Не было: сравнения тарифов, цен, выбора способа оплаты, визуального различия между планами.

**Контекст:**
- Telegram Mini App (Preact) + Telegram-бот
- Два способа оплаты: Telegram Stars (50 Stars / 30 дней) и банковская карта через Telegram Payments (299 ₽ / 30 дней)
- Три статуса подписки: `free`, `trial`, `active`
- Оплата происходит через бота (команды `/subscribe` и `/subscribe_stars`)
- Интерфейс двуязычный (ru/en)

## Что было сделано

### 1. Переработана секция подписки в Profile.tsx

**Бейдж текущего тарифа:**
- Визуально выделенный блок с иконкой (👑 Premium / ⏳ Пробный / 🆓 Бесплатный)
- Название текущего плана
- Дата окончания (для trial)

**Таблица сравнения тарифов (Free vs Premium):**
- Заданий в день: 3 / Безлимит
- Дуэли с друзьями: — / ✓
- Детальная статистика: — / ✓
- Все достижения: — / ✓

**Блок цен:**
- 50 Stars / 30 дней (Telegram Stars)
- 299 ₽ / 30 дней (банковская карта)

**Две кнопки оплаты:**
- «Оплатить Stars ⭐» → бот, команда subscribe_stars
- «Оплатить картой 💳» → бот, команда subscribe
- Для пользователей с активной подпиской ценовой блок и кнопки скрыты

### 2. Добавлены i18n ключи

Секция `subscription.*` добавлена в оба файла локализации (ru.json, en.json) — 22 новых ключа.

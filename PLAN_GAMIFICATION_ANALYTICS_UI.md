### 🗺️ ПЛАН ВЫПОЛНЕНИЯ

**Задача:** Геймификация, аналитика уровня пользователя и UI/UX-редизайн Brainify

**Тип:** Новая функциональность + рефакторинг UI

**Стратегия:** Три параллельных блока (сложность + геймификация, аналитика + когнитивный профиль, UI/UX-редизайн), объединённых в 10 последовательных этапов. Сначала — backend и данные (API, формулы, таблицы), потом — фронтенд, чтобы UI строился на уже работающих эндпоинтах. CSS-рефакторинг идёт параллельно. На каждом этапе существующая функциональность не ломается.

   - Рабочая гипотеза: вычислять на лету из последних попыток (5 штук), не хранить отдельно. При необходимости — кэшировать в Redis.

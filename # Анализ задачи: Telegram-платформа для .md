# Анализ задачи: Telegram-платформа для тренировки мозга

## 📋 ПОСТАНОВКА ЗАДАЧИ

**Название:** Telegram-платформа для тренировки мозга с социальными механиками

**Цель:** Создать продукт внутри экосистемы Telegram (Mini App + бот), который помогает пользователям тренировать когнитивные функции через мини-задания, удерживает через социальные механики (рейтинги, соревнования) и монетизируется подпиской. Виральный эффект — ключевой драйвер роста.

**Проблема:** Продукта не существует. Нужно спроектировать и реализовать с нуля. Ниша brain-training в Telegram практически пуста, при этом аудитория Telegram огромна и мультиязычна.

**Контекст:**
- Telegram Mini App — веб-приложение, запускаемое внутри Telegram, с доступом к Telegram API (авторизация, платежи, шаринг)
- Telegram-бот — дополняет Mini App: напоминания, быстрые задания в чате, уведомления о вызовах от друзей
- Единая экосистема: один аккаунт (Telegram ID), один бэкенд, два интерфейса
- Два языка — русский и английский — расширяют аудиторию и рынок

---

## 🎯 ГРАНИЦЫ (SCOPE)

### Входит:

**Telegram Mini App:**
- Авторизация через Telegram (без отдельной регистрации)
- Каталог мини-заданий по когнитивным категориям (память, внимание, логика, скорость)
- Интерактивное прохождение заданий (таймер, варианты ответа, ввод)
- Графики прогресса (динамика по дням/неделям, по категориям)
- Социальные функции: глобальный рейтинг, рейтинг среди друзей, PvP-соревнования (вызовы)
- Управление подпиской (пробный период → оплата через Telegram Payments)
- Мультиязычность: русский и английский

**Telegram-бот:**
- Ежедневные напоминания о тренировке
- Быстрое задание прямо в чате (1 задание без перехода в Mini App)
- Уведомления: «друг вызвал тебя на дуэль», «ты потерял позицию в рейтинге»
- Команда для приглашения друзей (реферальная ссылка)
- Мультиязычность: русский и английский

**Социальные и виральные механики:**
- Глобальный лидерборд (топ по очкам за неделю/месяц/всё время)
- Рейтинг среди друзей (контакты в Telegram)
- PvP-дуэли: один пользователь вызывает другого, оба решают одинаковое задание, сравниваются результаты
- Шаринг результатов: кнопка «Поделиться» после задания/дуэли (генерирует карточку с результатом для пересылки)
- Реферальная система: пригласи друга → получи бонус (дополнительные задания / продление пробного периода)
- Серии дней (streaks) с отображением в профиле

**Монетизация:**
- Пробный период с полным доступом
- Платная подписка через Telegram Payments
- Бесплатный режим — ограниченное количество заданий в день

### Не входит:
- Отдельный веб-сайт вне Telegram
- Мобильные нативные приложения (iOS/Android)
- Другие мессенджеры
- Командные соревнования (турниры, лиги) — рассмотреть в следующих итерациях
- Генерация заданий через AI в реальном времени
- Админ-панель для управления контентом (в MVP — задания из предзаполненной базы)
- Языки помимо русского и английского

## ⚠️ ОГРАНИЧЕНИЯ

- **Telegram Mini App:** работает внутри WebView Telegram; производительность зависит от устройства; ограничения по доступу к системным API
- **Telegram Payments:** поддерживает ограниченный набор провайдеров; комиссия; не все страны поддерживаются одинаково
- **Бот в чате:** UI ограничен inline-кнопками, текстом, изображениями — сложные задания только в Mini App
- **Мультиязычность:** весь контент заданий нужен на двух языках — это удваивает объём контента
- **PvP-дуэли:** требуется real-time или near-real-time механика сопоставления результатов
- **Рейтинг среди друзей:** Telegram не даёт доступ к контактам пользователя напрямую — друзья определяются по тем, кто тоже пользуется ботом

## 🔍 ФАКТЫ И ПРЕДПОЛОЖЕНИЯ

### Факты (подтверждённое):
- Платформа: Telegram Mini App + Telegram-бот (без отдельного сайта)
- Языки: русский и английский
- Социальные функции: рейтинги и соревнования — обязательны
- Виральный рост — приоритетная цель
- Модель монетизации: пробный период → подписка
- Мини-задания для развития когнитивных функций
- Графики для отслеживания прогресса

### Предположения (требует проверки):
- Основные виральные петли: PvP-дуэли (приглашение → вызов) и шаринг результатов
- Для дуэлей достаточен асинхронный режим (не требуется одновременная игра в реальном времени)
- Рейтинг друзей строится из пользователей приложения, которые пригласили друг друга по реферальной ссылке или обменялись вызовами
- Определение языка — по настройкам Telegram пользователя, с возможностью переключения
- Пробный период — 7 дней
- Бесплатный режим после пробного: 2–3 задания в день, без PvP-дуэлей, без детальных графиков
- Реферальная система: за каждого приглашённого — +3 дня к пробному периоду или +N бесплатных заданий

## 🚧 РИСКИ

| Риск | Вероятность | Влияние | Митигация |
|------|-------------|---------|-----------|
| Виральные механики не сработают — пользователи не делятся и не вызывают друзей | Средняя | Высокое | A/B тесты форматов шаринга; минимизация трения (один тап для вызова); ценные награды за приглашения |
| Дуэли асинхронны — второй игрок не отвечает, опыт ломается | Высокая | Среднее | Таймаут на ответ (24 часа); возможность играть с рандомным соперником; уведомления через бота |
| Telegram Mini App тормозит на слабых устройствах | Средняя | Среднее | Лёгкий фронтенд; минимум анимаций; progressive enhancement |
| Контент на двух языках сложно поддерживать и расширять | Средняя | Среднее | Структура данных с i18n с самого начала; задания с минимальным текстом (числа, паттерны) |
| Telegram Payments не доступен в нужных регионах | Низкая | Высокое | Предусмотреть альтернативный способ оплаты (Stars, внешний процессинг) |
| Злоупотребление пробным периодом через новые Telegram-аккаунты | Средняя | Низкое | Привязка к Telegram ID; ограничение по возрасту аккаунта; device fingerprinting |
| Недостаточный объём контента — пользователи быстро исчерпывают задания | Средняя | Высокое | Процедурная генерация вариаций (разные числа, позиции); параметризованные задания |

## 📌 ДОПУЩЕНИЯ

- Telegram Mini App — адекватная платформа для интерактивных заданий (WebView достаточно производителен)
- Telegram Payments покрывает ключевые рынки (РФ, СНГ, Европа)
- Дуэли работают асинхронно: один бросает вызов, второй отвечает в удобное время (до 24 часов)
- Для MVP достаточно 5–7 типов заданий с параметрической вариативностью (каждый тип генерирует множество комбинаций)
- Определение «друзей» = пользователи приложения, которые пригласили друг друга по реферальной ссылке или обменялись вызовами
- Telegram Stars может использоваться как альтернатива/дополнение к Telegram Payments

## ❓ ОТКРЫТЫЕ ВОПРОСЫ

| Вопрос | Рабочая гипотеза |
|--------|-----------------|
| Какие конкретно типы заданий? | Таблицы Шульте, запоминание последовательности, арифметика на время, Stroop-тест, поиск паттерна, числовой ряд, пары (memory game). Все параметризуются для бесконечной вариативности |
| Дуэли синхронные или асинхронные? | Асинхронные: вызов → соперник решает когда удобно (до 24ч) → сравнение результатов. Проще в реализации, не требует WebSocket |
| Стоимость подписки? | Не влияет на архитектуру, но нужно до запуска. Гипотеза: $2.99/мес или 249₽/мес |
| Telegram Stars или Telegram Payments? | Поддержать оба: Stars для микроплатежей/бустов, Payments для подписки |
| Как определять друзей? | Через реферальную систему: «друг» = тот, кого ты пригласил или кто пригласил тебя. Дополнительно — через PvP-вызовы |
| Нужна ли система достижений / бейджей? | Да, минимальная: «7 дней подряд», «100 заданий», «Победа в 10 дуэлях» — усиливает retention и даёт контент для шаринга |
| Какой формат карточки для шаринга? | Сгенерированное изображение с результатом + deeplink на бота для вовлечения нового пользователя |

## 📦 ГОТОВО ДЛЯ ПЛАНИРОВЩИКА

**Тип задачи:** Новая функциональность (greenfield-проект)

**Приоритет (оценка):** Высокий

**Зависимости:**
- Определение и наполнение контента заданий (минимум на двух языках)
- Регистрация бота и Mini App в Telegram (BotFather)
- Подключение Telegram Payments / Stars
- Выбор технологического стека (бэкенд, БД, фронтенд Mini App)

**Критерий готовности:**
1. Пользователь открывает Mini App через бота, авторизуется автоматически по Telegram ID
2. Доступно ≥5 типов заданий, задания можно проходить и получать оценку
3. Прогресс отображается на графиках (динамика, категории)
4. Работает глобальный лидерборд и рейтинг среди друзей
5. Можно вызвать друга на дуэль, оба решают, результат сравнивается
6. Шаринг результата/вызова отправляет сообщение с deeplink
7. Реферальная система отслеживает приглашения и начисляет бонусы
8. Пробный период 7 дней → ограничение → оплата через Telegram Payments
9. Интерфейс работает на русском и английском, язык определяется автоматически
10. Бот отправляет напоминания и уведомления о вызовах
